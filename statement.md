Expense Tracker

Problem statment:-
Managing personal finances is a critical everyday task, yet many individuals strugglee to track daily expenses and income accurately.
Without a propper tracking system, users may lose control of their spending habits, forget important transactions, or fail to maintain a clear record of financial activities.

A simple, affordable, and user-friendly solution is needed to help users monitor and manage their financial transactions efficiently.

Scope of the  project:-
1. Providing a CLI based application for managing expenses
2. Allowing users to add, view, edit, and delete financial records.
3. Saving all the records locally using JSON, ensuring no data is lost between sessions

Target Users:-
1. Students wanting to manage monthly pocket money and bill payments
2. Anyone needing a simple digital replacement for expense tracking.
3. Beginner developers learning file handling and CLI design in Python

High level Features:-
1. Add expense: User input amount, category,data, and description.
2. View Traction History: Shows all records.
3. Edit and delete Records: Modifyincorrect entries or remove outdated ones.
4. JSON Based Storage: Data is stored persistently without requiring a database.
