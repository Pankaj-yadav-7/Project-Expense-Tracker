Expense tracker

Overview of the project:- Expense tracker is a Python based command line application designed to help users manage their personal finances. It allows user to track daily expenses while maintaining a real time balance record. All data is stored locally using JSON file, making it simple and portable.



Features:- 

1.	Add expenses

2.	View all transactions anytime

3. 	Edit or delete existing records

4.	Data saved using JSON

5.  Easy to use CLI interface



Technologies/ Tools used:-

1. Python 3.x - Main programming language

2. JSON file handling - Data storage

3. datetime module - Recording dates



Steps to install:- 

1.	Download all the files in the same folder

2.	Open terminal in that folder

3.	Type “python main.py”

4.	Now you can use it freely and all the data will be stored in the same folder



Future Enhancements:-
